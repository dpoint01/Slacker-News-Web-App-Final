Slacker-News-Web-App-Final
==========================

Simple web app recreating the already existing Hacker News website (https://news.ycombinator.com) Now allows user to submit an article


This web app retrieves info from the Hacker News API and recreates another version of it called "Slacker News".

The goal of this Slacker News project was to play with HTML and CSS on a styling viewpoint and to also explore POST and GET
HTTP requests. 

This web app runs using sinatra locally. Instructions to run web app in the top of server.rb

Screen Shots included in repo.
